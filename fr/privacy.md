> Dernière mise à jour: **10 février, 2026**

La présente Politique de confidentialité décrit la manière dont les informations sont traitées lorsque vous visitez iscoffeetho.github.io (le «Site»).

## 1. Vue d’ensemble

Le Site est un site web statique hébergé via GitHub Pages. Il ne nécessite pas de comptes utilisateurs et ne collecte pas intentionnellement d’informations
personnelles.

## 2. Informations collectées automatiquement

Bien que le Site lui-même ne collecte pas de données personnelles, GitHub, en tant que fournisseur d’hébergement, peut collecter automatiquement certaines
informations techniques lorsque vous visitez le Site, notamment:

- Adresse IP
- Type et version du navigateur
- Système d’exploitation
- Pages de provenance
- Date et heure d’accès

Ces données sont traitées par GitHub conformément à sa propre politique de confidentialité et ne sont ni directement accessibles ni contrôlées par le
propriétaire du Site.

La politique de confidentialité de GitHub est disponible à l’adresse suivante:  
[https://docs.github.com/fr/site-policy/privacy-policies/github-privacy-statement](https://docs.github.com/fr/site-policy/privacy-policies/github-privacy-statement)

## 3. Cookies

Le Site lui-même n’installe ni n’utilise de cookies.

Cependant, GitHub Pages peut utiliser des cookies ou des technologies similaires dans le cadre de l’infrastructure de sa plateforme. Toute utilisation de ce
type est régie par les politiques de GitHub.

## 4. Analyses

Le Site n’utilise actuellement aucun outil d’analyse personnalisé (tel que Google Analytics, Plausible ou équivalent).

## 5. Contenus et liens tiers

Le Site peut contenir des liens vers des sites web tiers (tels que des dépôts GitHub ou de la documentation externe). La présente Politique de confidentialité
ne s’applique pas à ces sites, et il vous est recommandé de consulter leurs politiques de confidentialité respectives.

## 6. Sécurité des données

Étant donné que le Site ne collecte ni ne stocke de données personnelles, aucune donnée utilisateur n’est conservée par le propriétaire du Site. GitHub est
responsable de la sécurité de l’infrastructure d’hébergement.

## 7. Informations concernant les enfants

Le Site ne s’adresse pas aux enfants de moins de 13 ans et ne collecte pas sciemment d’informations personnelles auprès d’enfants.

## 8. Modifications de la présente politique

Cette Politique de confidentialité peut être mise à jour de temps à autre afin de refléter des changements apportés au Site ou à l’environnement d’hébergement.
Toute mise à jour sera publiée sur cette page avec une date de «Dernière mise à jour» révisée.

## 9. Contact

Si vous avez des questions concernant cette Politique de confidentialité, vous pouvez contacter le propriétaire du Site via le profil GitHub associé à ce site:

[https://github.com/IsCoffeeTho](https://github.com/IsCoffeeTho)
